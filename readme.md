# ntchffs-overlay

![Release](https://img.shields.io/github/v/release/notch4ff4/ntchffs-overlay)
![Downloads](https://img.shields.io/github/downloads/notch4ff4/ntchffs-overlay/total)

Данный плагин представляет собой ультимативный оверлей для OBS Studio, напоминающий всем известный Nvidia ShadowPlay, только ***на стероидах***.

# Возможности
## Основной функционал
Базово, оверлей повторяет все те же функции, что предоставлены в всем привычном оверлее от Nvidia, а именно:
- включение/выключение записи;
- включение/выключение/сохранение мгновенного повтора;
- индикация состояний.
## Галерея
Основная идея галереи состоит в том, что Вы не отрываясь от игры или работы, можете сразу же приступить к черновому монтажу или быстро обработать запись для отправки кому-то (обрезать, сжать, …). Для этого представлены все необходимые инструменты. Подробнее о том как всё это это работает с примерами использования показаны в *coming soon*.
## QoL настройки
Кроме обычных настроек, оверлей так же содержит в себе параметры, которые могут кардинально поменять опыт использования оверлеем:
- Умный реплей: повтор в OBS Studio немного отличается от стандартного повтора Nvidia. При повторном сохранении буфер не очищается от старого фрагмента записи. Это чревато тем, что в несколько сохраненных повторов за короткий промежуток времени, может попасть один и тот же фрагмент, что на этапе монтажа иногда мешает. Включение параметра умного повтора исправляет эту особенность – при сохранении буфер быстро выключается и включается обратно. Но в этом способе так же есть недостаток. Все, что произошло во время сохранения файла повтора так же не попадёт в новую запись (то есть, потеря нескольких секунд), для кого-то это может быть критично.
- Перехват фокуса: при включенном перехвате, оверлей работает как обычный Shadow Play. То есть, при отображении оверлея ты не можешь взаимодействовать с другим окном, мышь “принадлежит” только ему. Этот параметр просто необходим для игр, где мышь управляет камерой, ведь из-за особенности реализации мышь зачастую будет постоянно “телепортироваться” в центр, что попросту не даст без перехвата фокуса удобно пользоваться оверлеем. Но, если вдруг вы играете в игру, где управление не завязано на мыши и в целом Вы хотите совместить управление игрой и оверлеем – можете отключить этот параметр.
- Защита для OLED: так как индикаторы изначально белые – это создаёт проблему для дорогих OLED мониторов, а именно выгорание. С включением этого параметра мы делаем использование индикаторов безопасными, так как они начинают двигаться с какой-то периодичностью и убирают яркость.

# Установка
## Windows

Скачайте последнюю версию из раздела [Releases](https://github.com/notch4ff4/ntchffs-overlay/releases). Доступны два варианта:

| Файл | Для чего |
| ---- | -------- |
| `ntchffs-overlay-*-windows-x64-Installer.exe` | Обычная установка OBS (через установщик) |
| `ntchffs-overlay-*-windows-x64-portable.zip` | Portable-версия OBS |

Перед установкой закройте OBS Studio.

### Обычная установка (Installer)

1. Запустите `ntchffs-overlay-*-windows-x64-Installer.exe`.
2. Выберите тип установки:
   - **Standard OBS installation** — рекомендуется для обычной OBS. Плагин ставится в `C:\ProgramData\obs-studio\plugins\ntchffs-overlay\`, независимо от того, где установлен `obs64.exe`.
   - **Portable OBS installation** — для portable-версии. Укажите корневую папку OBS (ту, где лежат `bin`, `data` и `obs-plugins`). Установщик попробует найти OBS автоматически.
3. На следующем шаге при необходимости измените путь установки.
4. Перезапустите OBS Studio.
5. Настройте горячие клавиши для открытия оверлея и сохранения мгновенного повтора в настройках OBS Studio.

### Portable OBS (ZIP)

1. Распакуйте `ntchffs-overlay-*-windows-x64-portable.zip`.
2. Скопируйте папки `obs-plugins` и `data` из архива в корень вашей portable-установки OBS (туда, где лежат `bin`, `data` и `obs-plugins`).
3. При необходимости подтвердите замену существующих файлов.
4. Перезапустите OBS Studio.
5. Настройте горячие клавиши в настройках OBS Studio.

Структура после распаковки в корень portable OBS:

```
OBS-Studio\
  obs-plugins\64bit\ntchffs-overlay.dll
  data\obs-plugins\ntchffs-overlay\locale\...
```
